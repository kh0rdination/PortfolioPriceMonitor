# PortfolioPriceMonitor
This project looks up the latest prices on your asset portfolio for you using historical data provided online. 

UiPath Community edition .xaml files. Use the Community studio and a developer bot to test. 

Requirements: 2 excel/workbook sheets (/tmp and /Portfolio)

Portfolio worksheet is formatted like this: 

enter the prices you paid for assets in your portfolio, and calculate the difference on a daily basis. (much more sophisticated analysis can be made within excel.)
